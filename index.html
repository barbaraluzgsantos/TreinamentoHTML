    BDD Gherkin

    Anotações e curiosidades
    1. BDD é uma técnica de desenvolvimento de software ágil, criado por Dan North, em 2003, com o intuito de melhorar a comunicação entre um time de desenvolvimento (PO, QA e DEV) para que todos tenham o mesmo entendimento sobre uma determinada funcionalidade.
    2. O Gherkin é um dos elementos principais quando se trata de BDD em automação. Sua função é padronizar a forma de descrever especificações de cenários, baseado na regra de negócio.
    3. No Gherkin existem palavras chaves a serem utilizadas para especificar a forma como cada etapa funciona. As mais utilizadas são:
      1. GIVEN: Dado que - Indica o cenário inicial do teste
      2. WHEN: Indica uma ação que vai ser executada
      3. THEN: Indica o que aconteceu ao final da ação

    Páginas em HTML

    Anotações e curiosidades
    1. O HTML funciona com Tags, geralmente com abertura e fechamento < > e < / > .
    2. Estrutura de uma página HTML:
      1. < html > < /html >Esta tag é o elemento básico da estrutura do HTML. Assim sendo, ela conterá todos os elementos do seu documento. Todo documento HTML deve iniciar e finalizar com essa tag;
      2. < head > < /head > - Essa tag delimita o cabeçalho do documento. Não possui nenhum valor visível, porém é capaz de transmitir ao navegador diversas informações muito úteis e essenciais a uma boa apresentação do seu documento HTML;
      3. < title > < /title > - Essa tag define o título da sua página, o nome que aparecerá na sua aba, janela ou guia. Por esta razão, a tag >title> é de grande importância para o SEO;
      4. < meta > < /meta > - Essa tag permite inserir metadados ao seu documento, no caso acima, a informação charset=”UTF-8″, que garante a compatibilidade do código com os caracteres de padrão latino americano;
      5. < body > < /body > - Finalmente, a tag que representa o corpo do documento. Em síntese, é nessa tag que todos os elementos visíveis do seu site devem ser inseridos.
    3. Principais Tags de uma página HTML:
      1. < h > < /h > - Temos as Tags h1, h2, h3, h4, h5 e h5 para destacar os títulos de uma página. Sendo o nível 1 o maior, e o nível 6 o menor em hierarquia. Usamos apenas um h1 para melhores práticas de SEO.
      2. < p > < /p > - Principal tag de texto, compõe um parágrafo;
      3. < span > < /span > - Apesar de ter uma funcionalidade e características parecidas com os parágrafos, costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc. Também pode ser utilizada para formar um container;
      4. < b > < /b > - Transforma o conteúdo em negrito;
      5. < i > < /i > - Transforma o conteúdo em itálico;
      6. < /br > - Essa tag não necessita de fechamento, ela executa a função de quebra de linha;
      7. < /hr > - Essa tag não necessita de fechamento, ela forma uma linha horizontal.
